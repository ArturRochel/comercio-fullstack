# 🧠 Projeto FullStack - Controle de Mercadinho

Este é o desenvolvimento do meu primeiro projeto fullstack, com foco em criar uma aplicação web para **controle de estoque e gerenciamento de produtos e preços** para um mercadinho familiar. A aplicação contará com uma interface amigável, voltada principalmente para dispositivos móveis, e será acessível por qualquer usuário através de um navegador, sem necessidade de instalação ou conhecimento técnico.

---

## 🎯 Objetivos do Projeto

- Criar uma aplicação completa com front-end e back-end separados.
- Permitir o cadastro, edição, listagem e remoção de produtos.
- Garantir persistência dos dados através de um banco de dados online.
- Focar em responsividade e facilidade de uso.
- Aprender e aplicar conceitos de API REST, CRUD, e hospedagem gratuita.

---

## 🧰 Tecnologias Utilizadas

### 🔧 Back-end

- **Node.js**
- **Express.js**
- **MongoDB (MongoDB Atlas)**
- **Mongoose**
- **Dotenv**
- **Cors**
- **Nodemon**
- **Hospedagem:** [Render](https://render.com)

---

### 🎨 Front-end

- **React.js**
- **Tailwind CSS** (opcional para estilização)
- **Axios**
- **React Router DOM**
- **React Hook Form**
- **Hospedagem:** [Vercel](https://vercel.com) ou [GitHub Pages](https://pages.github.com)

---

### 🗃️ Banco de Dados

- **MongoDB**
- **Hospedagem:** [MongoDB Atlas](https://www.mongodb.com/atlas)
